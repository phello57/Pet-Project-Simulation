Выключить логи в main.rb - второй арг - action = Actions.new(game_map, true)

медведь охотится на свинью
свинья на траву
есть камни и деревья, которые обходят свинья и деревья

settings.rb - настройка карты, спавна и характеристик существ
render.rb - рендер карты
game_map.rb - карта с методами создания
actions - действия над картой
bfs - вся карта это двумерный массив, в каждой клетке лежит Node и она связана с 8 Node вокруг неё
поиск в ширину ищет ближайшую подходящую цель для существа по @set_goals, которые задаются при инициализации
сущности
поиск в ширину на каждой итерации от цели создает PathNode - это связь предыдущей ноды и следующей, что бы мы
могли потом следовать этому пути

есть стамина у сущности, когда стамина == 0, то начинает падать хп

если у сущности хп == 0 то она пропадает с карты
